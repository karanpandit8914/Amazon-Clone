# 🛒 Amazon Clone Website

A simple **Amazon Homepage Clone** built using **HTML and CSS**.
This project replicates the layout and design of the Amazon landing page and was created to practice and demonstrate fundamental **front-end development skills**.

---

## 🌐 Live Demo

🔗 **View Website:** https://karanpandit8914.github.io/Amazon-Clone/

---

## 📌 Project Overview

This project recreates the **UI structure of Amazon's homepage**, including the navigation bar, hero section, product category sections, and footer.

The goal of this project was to strengthen understanding of **HTML page structure and CSS styling**, while building a real-world inspired interface.

---

## ✨ Features

* Amazon-style **navigation bar**
* **Search bar interface**
* **Hero / banner section**
* **Product category cards**
* **Structured layout similar to Amazon**
* **Footer section**
